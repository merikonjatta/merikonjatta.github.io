JekyllのLiquidテンプレート内でpryする

Liquidテンプレートはデバッグしにくい。どんな変数が使えて、どんなデータ構造になっているのか、そのあたりの透明性が低い。{% binding.pry %} しようとするとそんなタグねーよバカ！カス！ハゲ！と怒られる。

そんなタグを定義すれば良いだけ。

# _plugins/pry.rb
class LiquidPryTag < Liquid::Tag
  def initialize(tag_name, markup, tokens)
    super
  end

  def render(context)
    context.pry
  end
end

Liquid::Template.register_tag("pry", LiquidPryTag)
